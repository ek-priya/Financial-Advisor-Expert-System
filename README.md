
# Financial Advisor Expert System


## Overview
The Financial Advisor Expert System is a rule-based intelligent system designed to provide personalized financial advice based on user inputs such as income, savings, risk tolerance, and financial goals.

This project simulates how human financial advisors make decisions by using predefined rules and logic.



##  Problem Statement
Many individuals lack access to professional financial guidance due to:

 * High consultation costs 💸
* Lack of financial awareness 📉
* Difficulty in understanding investment options

This system aims to provide basic financial guidance automatically.


##  Solution
The system takes user inputs and:

* Analyzes financial condition
* Evaluates risk appetite
* Suggests suitable investment strategies

It acts as a virtual financial advisor.


##  Features

* 📌 User input-based decision making
* 📊 Risk profile classification (Low / Medium / High)
* 💰 Investment recommendations
* 🧠 Rule-based expert system logic
* 📓 Implemented using Jupyter Notebook
